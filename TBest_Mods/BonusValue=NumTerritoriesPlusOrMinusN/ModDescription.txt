Each bonus is by default set according to the following formula.

BonusValue = numTerriotries + N

Typicaly N is -1 for the many of the competative templates. This mods makes it easy to make this the case on all maps.

Additionaly there are checkboxses to alter what formula is used. 
Specsifically it can also change all bonus by N, multply by N using either the default bonus value or the number of territories in that bonus. 

Bonuses that are 0 or negative can be excluded from the mod the effects of this mod.
